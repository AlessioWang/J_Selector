# J_Selector

A 3D Selector for Java and Processing

All check shape elements should be added in Container.

Tips: There is a demo in Demo folder.
## Support element:
- WB_Polygon
- He_mesh

## Dependencies:
- He_mesh
- Guo_Cam
- Processing
